# 🎬 All Video Downloader - Usage Instructions

## ⚠️ Windows SmartScreen Warning Solution

When you run the executable for the first time, a Windows SmartScreen warning may appear. This is normal and is a security measure.

### 🔧 Solution Steps:

1. **Click "More info" link**
2. **Click "Run anyway" button**
3. The program will open

### 📋 Alternative Solutions:

#### Method 1: Right-Click to Run
- **Right-click** on the executable
- Select **"Run as administrator"**
- No warning will appear

#### Method 2: Add Exception to Windows Defender
1. Open **Windows Security**
2. **Virus & threat protection** → **Manage settings**
3. **Exclusions** → **Add an exclusion**
4. Select **File** and choose the `URLConverter.exe` file

#### Method 3: Run with PowerShell
```powershell
# In the executable's folder:
Unblock-File -Path "URLConverter.exe"
.\URLConverter.exe
```

## 🚀 Usage

1. **Enter URL** - Paste the video link
2. **Select Format** - MP4 (video) or MP3 (audio)
3. **Select Quality** - Choose one of the 1080p options
4. **Select Folder** - Determine the download location
5. **📥 DOWNLOAD** - Download starts

## 🔧 Troubleshooting

### Video Not Downloading:
- Check your internet connection
- Make sure the URL is correct
- Temporarily disable your antivirus program

### Low Quality:
- Install FFmpeg (optional)
- Select a higher quality option

### Program Won't Open:
- Add exception to Windows Defender
- Run as administrator
